**Script collection for Adobe Illustrator, bundled as a CEP extension.**






















## Development



Setting up:
```bash
git clone git@github.com:jaid/emote-workflow.git
cd emote-workflow
npm install
```
Testing:
```bash
npm run test:dev
```
Testing in production environment:
```bash
npm run test
```
